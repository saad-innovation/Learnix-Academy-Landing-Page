# Learnix Academy - Landing Page

Learnix Academy aik modern educational landing page hy jo students ko professional aur interactive environment provide karne ke liye design kiya gaya hy. Yeh project mere frontend development portfolio ka aik aham hissa hy.

## 🚀 Live Demo
Aap is website ko yahan live dekh sakte hain: 
**[Live Demo Link Yahan Dalein]** (GitHub Pages ka link yahan paste karein)

## ✨ Key Features
- **Modern & Vibrant UI:** Poster-inspired energetic colors (Cyan and Gold) ke sath professional look.
- **Fully Responsive:** Laptop, Tablet, aur Mobile screens ke liye optimized layout.
- **Interactive Navigation:** Mobile devices ke liye functional hamburger menu.
- **Smooth Scrolling:** User-friendly experience ke liye sections ke darmiyan smooth transitions.
- **Call to Action (CTA):** High-conversion "Admission Open" aur "Enroll Now" sections.

## 🛠️ Technologies Used
- **HTML5:** Semantic structure ke liye.
- **CSS3:** Custom styling, Flexbox, aur CSS Grid layouts ke liye.
- **JavaScript (ES6):** DOM manipulation aur navigation interactivity ke liye.
- **FontAwesome:** Professional UI icons ke liye.
